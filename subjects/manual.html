📋 概览 Overview

Round 1 是一对一书面答题环节。学生看裁判设备上的题目，将答案手写到纸质答题卡上，裁判核对答题卡与系统答案，判定正确/错误后提交记录。

Round 1 is a one-on-one written round. The student views the question on the judge's device and writes their answer on a paper answer card. The judge checks the card against the system answer, marks correct/incorrect, and submits.

项目说明
题型Fill-in-Blank（填空）/ MCQ（仅 CHI 科目）
答题方式学生看裁判设备，手写到纸质答题卡
每人次1 题，提交后自动返回科目选择
记录方式Supabase 上传 + 本地 JSON 文件下载

⚠️ 注意事项 Notes
!

常见问题与处理方式

Common issues and how to handle them

▾
📶

断网：系统自动切换至本地题库，答题与提交仍可进行，但 Supabase 上传会失败——JSON 文件此时尤为重要，请务必保留。
If offline, the app falls back to the local question bank. Keep your JSON files — they're the only record.

🚫

不要刷新页面：刷新会重置当前题目和会话状态。如误刷新，重新选班级→选科目即可继续。
Don't refresh — it resets the question. If refreshed, just reselect class → subject.

🔄

题目重复：系统尽量避免重复，但题库较小时可能出现。可点击「↺ New Question」跳过，不计入记录。
The system avoids repeats, but may repeat on small pools. Tap "New Question" to skip — no record is created.

📥

下载弹窗：部分手机浏览器在下载 JSON 时会弹出确认窗，请点击「允许」或「保存」，确保文件成功下载。
Some mobile browsers prompt you to confirm downloads — tap Allow/Save to ensure the file is saved.

📋 概览 Overview

Round 2 是 60 秒限时选择题环节。裁判与学生共同看屏幕，学生读完题目后说出 A/B/C/D，裁判等学生报答案后再点击对应选项，系统自动判断并进入下一题。

Round 2 is a 60-second timed MCQ round. The judge and student view the screen together. After the student reads the question and calls out A/B/C/D, the judge taps the corresponding option and the system auto-scores and advances.

题型MCQ 四选一 (A/B/C/D)
时间60 秒自动倒计时
计分答对 +10 分，答错 0 分（不扣分）
操作分工学生读题并报答案，裁判点击选项

⚠️ 注意事项 Notes
!

常见问题与处理方式

Common issues and how to handle them

▾
📶

断网：断网时 Supabase 上传会失败，但 JSON 文件仍会正常下载到本地——JSON 文件此时是唯一备份，请务必保留。
If offline, Supabase upload fails but the JSON file will still download locally — keep it as the backup.

🙈

防止抢跑：进入答题页后先让学生准备好，再点击 Start。学生看到屏幕时计时才正式开始，确保公平。
Let the student get ready before tapping Start — the timer begins only when you tap, ensuring fairness.

👂

听清再点：学生报答案后，裁判需稍停确认再点击，避免因未听清而误操作。若学生改口，以最终报出的答案为准。
Wait a moment after the student calls their answer before tapping. If the student changes their answer, take the final answer.
